# 🎮 Number Guessing Game

A simple Python mini project where the computer generates a random number and the player tries to guess it.

## ✨ Features

* Random number generation
* Three difficulty levels:

  * Easy (1–50)
  * Medium (1–100)
  * Hard (1–500)
* Unlimited guesses
* Too High / Too Low hints
* Attempt counter

## 🛠️ Technologies Used

* Python 3.11

## ▶️ How to Run

```bash
python main.py
```
