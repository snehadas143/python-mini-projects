# 🔐 Password Generator

A Python-based password generator that creates strong and secure passwords based on user preferences.

## ✨ Features

- Generate passwords of custom length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special characters
- Password strength indicator
- Copy password to clipboard
- Save generated passwords to a file

## 🛠️ Technologies Used

- Python 3
- random
- string
- pyperclip

## 📂 Project Structure

```
02_password_generator/
│── main.py
│── README.md
│── passwords.txt
```

## ▶️ How to Run

1. Install Python 3.
2. Install pyperclip:

```bash
pip install pyperclip
```

3. Run the program:

```bash
python main.py
```
